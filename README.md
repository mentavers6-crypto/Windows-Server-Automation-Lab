# Windows Server 2022 Automation Lab

## Project Overview
This project demonstrates the automation of a Windows Server infrastructure using PowerShell. It was designed during my studies at **CMC Agadir**.

## Features
- **Active Directory**: Automated Forest and Child Domain (`agadir.cmc.local`) setup.
- **DHCP**: Automatic scope configuration and authorization.
- **User Management**: Creation of OUs (IT, Gestion), Security Groups (RH, DEV), and users with specific permissions.
- **File Server**: Secure SMB shares with NTFS permission validation.
